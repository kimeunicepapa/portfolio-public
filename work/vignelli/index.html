The Vignelli Canon

Graphic designer Massimo Vignelli writes a guide to good design.
ROLE Graphic Designer
YEAR 2021
DURATION 4 weeks
TOOLS Adobe InDesign

In The Vignelli Canon, Massimo Vignelli proposes a set of rules that should be taken into consideration when talking about good design. The publication design for this book emphasizes Vignelli’s message about design — clarity, appropriateness, and timelessness. The redesign incorporates a clear type hierarchy, bold titles, and dynamic compositions.

ABOUT

Refined taste, a curious mind, consistent work, and a deep obsession with the craft.
