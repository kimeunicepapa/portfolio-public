The Vignelli Canon

OVERVIEW

In The Vignelli Canon, Massimo Vignelli proposes a set of rules that should be taken into consideration when talking about good design. The publication design for this book emphasizes Vignelli’s message about design — clarity, appropriateness, and timelessness. The design incorporates a clear type hierarchy, bold titles, and dynamic compositions.
